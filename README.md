# gulp_ejs_SASS_template


### Initialization
initialize this project with: 
```
npm init -y
```
then: 
```
npm install --save-dev gulp gulp-babel @babel/core @babel/preset-env gulp-connect gulp-ejs gulp-imagemin gulp-rename gulp-sass gulp-sourcemaps node-sass
```
